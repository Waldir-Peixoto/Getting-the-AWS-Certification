O que é inteligencia artificial?

IA - comportamento inteligente das maquinas
Imitação de habilidades cognitivas dos seres humanos. 
A IA se baseia em Engenharia do Conhecimento. 
Ela deve resolver o problema como um ser humano resolveria. 
O objetivo é transferir conhecimentos humanos para a maquina. 
Esse processo de alimentação de conhecimentos na maquina é chamado de modelagem. 
Machine learning =- nos dizemos a maquina a diferenca entre dois modelos (quadrados e circulos)
Deep Learning - Nos jogamos varios dados, e o proprio software comeca a examinar os dados e fazer inferencias dos objetos.
